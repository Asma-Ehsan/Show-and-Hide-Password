# Show and Hide Password

A simple and beginner-friendly JavaScript project to toggle password visibility and practice core JS concepts.

## Live Demo
Try it live here: [Show and Hide Password](https://asma-ehsan.github.io/Show-and-Hide-Password/)

## Features
- Toggle password visibility (show / hide) with a single click.
- Interactive eye icon that changes based on password state.
- Demonstrates use of:
  - DOM selection: `let password = document.getElementById("password");`
  - Event handling: `eyeicon.onclick = function() { ... }`
  - Changing input type dynamically: `password.type = "text"` or `"password"`

## How to Use
1. Open `index.html` in your browser.
2. Type a password in the input field.
3. Click the **eye icon** to show or hide the password.
